SARAH - Medicina, Tecnologia e Bem-Estar

O projeto tem como objetivo fornecer uma plataforma para o gerenciamento de consultas de uma Clínica.

Endpoints:

  • Listagem de Médicos
  • Listagem de Convênios
  • Logar na plataforma
  • Cadastro na plataforma
  • Marcar uma consulta
  • Ver detalhes de uma consulta
  • Remover consulta
  • Reagendar uma consulta
